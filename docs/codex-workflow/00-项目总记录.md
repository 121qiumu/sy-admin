- [04-首批开发模块规划.md](./04-首批开发模块规划.md)

关键结论：

- 管理后台接口统一前缀：`/admin/...`
- token 通过请求头 `Authorization` 传递
- token 值直接传原始字符串，不带 `Bearer`
- 成功业务码：`1000`
- 通用分页结构：`list + pagination`
- 默认分页大小：`15`
- 上传接口使用 `multipart/form-data`
- 上传成功后 `data` 直接返回文件 URL
